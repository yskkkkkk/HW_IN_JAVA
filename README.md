# HW_IN_JAVA

Java 기초·객체지향 실습과 알고리즘 문제 풀이를 기록한 개인 학습용 저장소입니다.

## 구성

소스는 `src/com/yskkkkkk/` 아래 학습 시기별 패키지로 정리되어 있습니다.

| 패키지 | 내용 |
|---|---|
| `july/` | Java 기초·객체지향 실습 — 입출력, 상속/다형성, 인터페이스와 DAO 패턴, XML 파싱(DOM/SAX), 오목 게임 등 |
| `august/` ~ `october/` | 백준(BOJ) 알고리즘 문제 풀이 — BFS/DFS, 유니온-파인드, 최소 신장 트리(Kruskal), DP, 2차원 배열 시뮬레이션, 스택/큐, 링크드리스트 |
| `august/pcs/` | 순열·조합·부분집합(permutation / combination / subset) 연습 |

## 파일 명명 규칙

- `Main_<문제번호>_<제목>.java` · `Solution_<제목>.java` — 백준 문제 번호와 제목을 그대로 보존해, 나중에 문제를 다시 찾기 쉽게 정리했습니다.
- 일부 `*.txt`는 해당 문제의 샘플 입력 데이터입니다.

## 환경
